Command 'git clone'

Der Befehl 'git clone' kopiert ein bereits exestierendes Repository um es lokal bearbeiten zu können.

How to use

Lokales Repo klonen:

git clone < Pfad - zum - Reop > [ Name des Verzeichnisses (optional) ]

Server Repo klonen:

git clone < Server - Name | url >: < Pfad - zum - Repo > [ Name des Verzeichnisses (optional) ]

Beispiel:

$ git clone git://github.com/juliane99/Gruppenaufgabe.git coolesVerzeichnis

Was macht git clone genau?

Auf das Beispiel bezogen, legt Git ein neues Verzeichnis namens "coolesVerzeichnis" an und kopiert alles aus dem Repo "Gruppenaufgabe" von juliane99 in dieses hinein. Wäre "coolesVerzeichnis" nicht angegeben, würde Git automatisch ein verzeichnis namens "Gruppenaufgabe" erstellen.
